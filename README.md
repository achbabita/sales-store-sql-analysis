# Sales Store Data Analysis using SQL Server

##  Overview

This project focuses on analyzing retail sales data using SQL Server and SSMS to uncover business insights related to customer behavior, product performance, revenue trends, cancellations, and purchasing patterns.

---

##  Tools Used

- SQL Server
- SQL Server Management Studio (SSMS)
- SQL

---

##  Key Analysis Performed

- Top selling products
- Most cancelled products
- Peak purchase hours
- Highest spending customers
- Revenue by product category
- Cancellation & return rate analysis
- Payment method analysis
- Customer age-group analysis
- Monthly sales trend analysis
- Gender-based purchasing behavior using PIVOT

---

##  SQL Concepts Used

- GROUP BY
- ORDER BY
- CASE WHEN
- Aggregate Functions
- DATEPART()
- FORMAT()
- PIVOT
- Conditional Counting
